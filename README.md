# AI Industry Research

聚焦 AI、AI 智能体、机器人、具身智能、AI 医疗、自动驾驶、AI 金融、AI 互联网、脑机接口 的行业研究、公司调研与产品调研。

**在线阅读**: https://jeffliulab.github.io/ai-industry-research/

## 姐妹站

- [AI Notes · 人工智能学习笔记](https://jeffliulab.github.io/ai-notes/) — 偏技术原理与课程学习
- AI Industry Research（本站）— 偏行业 · 公司 · 产品的调研笔记

## 本地预览

```bash
pip install -r requirements.txt
python hooks/generate_dropdown.py
mkdocs serve
```

打开 http://127.0.0.1:8000 查看。

## 内容组织

```
docs/
├── 01_AI/          # 大模型、多模态、AI 基础设施、地缘
├── 02_机器人/      # 人形机器人、产业链、核心零部件
├── 03_具身智能/    # VLA、World Model、Sim2Real
├── 04_AI金融/      # 量化、投研、风控、合规
├── 05_AI互联网/    # 搜索、浏览器、AI Coding、应用层
├── 06_AI医疗/      # 药物发现、影像、临床 Copilot
├── 07_自动驾驶/    # L2+ · L4 · Robotaxi
├── 08_脑机接口/    # BCI 技术路线、医疗/消费
├── 09_AI智能体/    # Coding Agent、Computer Use、Multi-Agent
└── 99_方法论/      # 调研框架与信息源
```

每个板块按"**内容维度**"扁平化组织（01-10 前缀），配合 `11_公司研究` / `12_产品研究` 作为实体索引。详见 `行业研究标准方法.md`。
