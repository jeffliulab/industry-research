# Arc Search / Dia（The Browser Company）

> 最后更新：2026-04-24
>
> Arc 是 The Browser Company 2022 年推出的**重设计浏览器**——**垂直标签栏 + 空间组织**让开发者 / 设计师狂热。**2024-02 Arc Search（iOS）**"Browse for me" 用 AI 总结搜索结果。**2024-10 宣布战略转向 Dia**（AI 原生浏览器，2025 公测）。

## 一、产品定位

Arc = **"为桌面重设计的浏览器"**（MAU 小但粉丝忠诚）
Arc Search = **"iOS 版 Arc + AI 搜索"**
Dia = **"AI 原生浏览器" Arc 继任者**（2024-10 宣布 → 2025 公测）

## 二、核心能力

### Arc
- 垂直标签 + Spaces
- Split View / Little Arc
- Easel / Notes

### Arc Search（iOS）
- **"Browse for me"**：输入问题 → AI 遍历网页 → 生成页面
- 彻底跳过"手动点击网页"

### Dia
- **AI 侧边栏原生**
- **Agent 任务框**
- 更精简 UI

## 三、版本演化

| 时间 | 里程碑 |
|---|---|
| 2022 | Arc 公开测试 |
| 2023 | Arc 1.0 Mac |

| 2024-Q3 | Arc Windows |
| **2024-10** | **Arc 转向 Dia** 战略 |
| 2025 | Dia 公测 |
