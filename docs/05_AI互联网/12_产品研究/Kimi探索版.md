# Kimi 探索版

> 最后更新：2026-04-24
>
> Kimi 探索版是 **Moonshot 2024-10 推出的"Deep Research"风格功能**——对标 Perplexity / ChatGPT Deep Research，让 Kimi 执行 **多步检索 + 深度报告**。**是中国 Deep Research 第一款公开产品**，2025 成为 Kimi 付费转化的重要驱动。

## 一、产品定位

Kimi 探索版 = **"中国版 Deep Research"** —— Moonshot 把长上下文能力（200 万字）+ Agent 多步检索 结合，产出 **15-30 页研究报告**。**Pro 订阅才能用**（¥199/月），**提升 Kimi ARPU**。

## 二、核心能力

- **多步骤 Agent 检索**：10-30 个 sub-query
- **200 万字长上下文**：全文档整合
- **结构化报告**：带引用 + 图表
- **时长**：5-20 分钟
- **中文优先**：中文语料深度
