# Perplexity Pro（含 Comet 浏览器）

> 最后更新：2026-04-24
>
> Perplexity Pro 是 Perplexity 的 $20/月订阅——**无限 Pro Search、模型切换（GPT/Claude/Grok）、Deep Research 无限、Comet 浏览器**。2025 Pro 付费用户 ~100 万，ARR ~$200M。**公司详见 [AI 板块](../../01_AI/11_公司研究/Perplexity.md)**。

## 一、产品定位

Perplexity Pro = **"知识工作者的 AI 搜索 + 浏览器一体化"** —— 不是简单订阅，而是**"Perplexity 全家桶"**：Pro Search / Deep Research / Pages / Comet / 模型切换。**$20/月对标 ChatGPT Plus 但更专注搜索 + 研究**。

## 二、核心能力

- **Pro Search**：多步骤自动搜索 + 引用
- **Deep Research**：10-20 步深度调研（2025 开放）
- **Pages**：AI 生成报告 / 博客
- **Comet 浏览器**：AI 原生 Chromium fork（2025 公开）
- **模型切换**：GPT-4o / Claude / Grok / Llama / Sonar（自研）

## 三、版本演化

| 时间 | 里程碑 |
|---|---|
| 2022-12 | Perplexity 首发 |
| 2023-09 | Pro $20/月订阅 |
| 2024 | Pro Search 2.0 |
| 2025-Q1 | **Comet 浏览器发布** |
| 2025-Q2 | **Deep Research GA** |
| 2025-Q3 | Pro 用户突破 100 万 |

## 四、定价与商业化

- **Pro**：$20/月 或 $200/年
- **Enterprise**：$40/seat
- 2025 ARR ~$200-250M
- Pro 付费率估算 5-8%（C 端 AI 平均水平）

## 五、用户反馈

- **"引用最准、界面最清爽"** —— 知识工作者首选
- **Comet 小众但狂热**
- 批：**广告实验** 让部分用户不满
