# 医疗影像 AI 格局

> 最后更新：2026-04-24
>
> 医疗影像 AI 是 **AI 医疗最成熟子赛道**——2014-2019 第一波泡沫（IBM Watson Health 等受挫），**2020-2026 第二波扎实落地**：**中国 3 强（推想 / 深透 / 商汤 Meera）+ 美国 Aidoc / Viz.ai / PathAI**。**FDA 批准 AI 医械累计 ~1000 个（2025），影像占 70%**。

## 一句话结论

医疗影像 AI 2026 状态：**已跨越鸿沟进入 Early Majority**，医院采用率 30-50%，主要场景 **CT / MRI / 病理 / 眼底**。**中国公司规模 + 数据 最大，美国公司合规 + 产品成熟度最高**。

## 三条关键要点

1. **Aidoc（以色列 / 美国）$1.2B 估值**：Stroke / 出血等紧急影像 AI 龙头
2. **中国医院采用率全球第一**：推想 / 深透 / 联影合占 70%+ 三甲医院
3. **PathAI / Tempus** 把 AI 从影像扩到**病理 + 精准医疗**

## 主要场景

### 放射影像
- **CT**：肺结节、中风、出血
- **MRI**：脑、心、骨
- **X 光**：胸部、骨折
- **乳腺 / Mammography**

### 病理
- **组织切片数字化** + AI 分析
- **PathAI / 华大 / Tempus**

### 眼底 / 视网膜
- **糖尿病视网膜病变**（DR）
- **IDx-DR** 是首个 FDA 批准自主 AI 诊断（2018）

### 皮肤
- **痣 / 皮肤癌**
- **Google Derm Assist** 等

### 超声 / 心电
- **Butterfly Network**（便携超声）
- **AliveCor KardiaMobile**

## 商业模式

- **RaaS / Per-scan 授权**（$1-10 per scan）
- **企业授权**（$100k-1M/年）
- **Medicare 报销**（2023 起部分覆盖）

## 监管

- **FDA**：已批准 ~1000 个 AI/ML 医械（2025）
- **NMPA**（中国）：二三类 AI 加速通道
- **CE Mark**（欧盟）：严格但标准化

## 挑战

- **医院部署慢**（IT + 采购）
- **收入天花板**（单产品 $50-200M ARR 常见）
- **数据 / 隐私 HIPAA 严格**

## 用 [跨越鸿沟 框架](../../99_方法论/框架_跨越鸿沟.md) 看

- **早期采用者（2016-2020）**：顶级医院试点
- **Early Majority（2020-2025）**：学术中心 + 大医院
- **Late Majority（2026+）**：社区医院 / 基层（AI 普惠)

## 2026 关键变量

1. **Aidoc / Viz IPO**：估值测试
2. **中国医疗影像 港股 / 美股 IPO 潮**
3. **FDA 自主 AI 诊断（无需人类放射科）扩展**
4. **大模型 + 影像 AI 融合**（Gemini Medical 等）

## 我的判断

> 1. **医疗影像 AI 已成熟**，但增速放缓
> 2. **中国公司规模最大但估值被限**（地缘 / 监管）
> 3. **PathAI / Tempus 等扩展到病理 / 基因** 是下一站
> 4. **LLM + 影像融合** 是 2026-2028 新机会
>
> **我可能错在哪里**：LLM 技术快速替代传统影像 AI。
