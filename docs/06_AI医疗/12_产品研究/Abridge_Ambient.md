# Abridge Ambient

> 最后更新：2026-04-24
>
> Abridge Ambient 是 Abridge 的**核心产品**—— "**AI 监听医患对话 → 自动生成 SOAP 病历 + 医嘱 + ICD 编码**"。**部署至 Kaiser、Emory、Yale、Cleveland Clinic** 等顶级医院系统。

## 一、产品定位

Abridge Ambient = **"医生口述 → 结构化病历" AI 标杆** —— **医生省时 30-60 分钟/天**是核心 ROI。**Epic / Cerner EMR 原生集成**。**公司详见 [Abridge 公司研究](../11_公司研究/Abridge.md)**。

## 二、核心能力

- **实时 ASR**（医学专用）
- **对话理解**（多说话人分离）
- **SOAP 病历结构化**：主诉 / 现病史 / 查体 / 评估 / 计划
- **ICD-10 编码建议**
- **医嘱草稿**
- **EMR 推送**（Epic / Cerner / Meditech）

## 三、工作流

```
医生进诊室 → 打开 Abridge App
医患对话（正常进行 10-30 分钟）
Abridge 实时转写 + 理解
对话结束 → 2 分钟后：
  - 病历草稿
  - 医嘱
  - 诊断代码
医生 5 分钟审核 → 推入 EMR
```

## 四、支持场景

- 初诊 / 复诊
- 远程医疗
- 急诊
- 专科门诊（心内 / 儿科 / 精神 / 等）

## 五、定价

- **Per-provider** $300-500/月
- **Per-encounter** $3-10
- **Enterprise 合同** $500k-10M ACV

## 六、代表客户

## 七、ROI

- **医生时间**：节省 30-60 分钟/天
- **病历质量**：Structured + ICD 精准
- **医生满意度 / 减少 burnout**
- **Payback** 3-6 个月
