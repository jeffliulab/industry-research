# Nabla Copilot

> 最后更新：2026-04-24
>
> Nabla Copilot 是 **欧洲法国 Nabla 公司的临床 Copilot 产品**——**与 Abridge Ambient 同类但欧洲原生**。**多语言（法 / 英 / 德 / 西等）+ GDPR 合规** 是差异化。**公司详见 [Nabla](../11_公司研究/Nabla.md)**。

## 一、产品定位

Nabla Copilot = **"欧洲版 Abridge"** —— 法国起家，**GDPR 原生 + 多语言**，在欧洲医院基本首选。**2024 进军美国市场**（Permanente Medicine 早期客户）。

## 二、核心能力

- 实时 ASR + 病历结构化
- **GDPR 原生**（数据存欧洲）
- **多语言 10+**（法 / 英 / 德 / 西 / 荷 / 意 等）
- **非诊断定位**（欧洲监管友好）

## 三、客户

- **Ramsay Santé**（欧洲最大医院集团）
- **法国多家公立 / 私立**
- **美国 Permanente Medicine**（Kaiser 子机构）

## 四、定价

- Per-provider €300-500/月
- Enterprise 合同
