# 临床 Copilot 全景：Abridge · Nabla · Epic AI

> 最后更新：2026-04-24
>
> 临床 Copilot（Ambient Clinical Documentation）是 **AI 医疗商业化最快赛道**——"**医生戴着/放着录音，AI 转文书**"节省每日 1-3 小时。2024-2026 爆发，**Abridge ARR 从 $50M → $500M+ 目标**（18 个月 10x），与 Microsoft Dragon Ambient、Epic AI、Suki 等构成 **百亿美元市场**。

## 一句话结论

临床 Copilot 是 **AI 医疗的"Cursor 时刻"**——产品化快 + ROI 清晰 + 医生欢迎。**Abridge 事实领先**，**Nabla 欧洲代表**，**Epic AI 凭借 EMR 分发**，**Microsoft Dragon Ambient 全球规模最大（10 万医生）**。

## 三条关键要点

1. **Abridge $2.7B 估值**：Big Health System 批量签约（Kaiser、Emory、Yale 等）
2. **Microsoft Dragon Ambient (Nuance 收购)** 仍有 10 万 + 医生用户
3. **Epic Ambient Voice** (2024) 集成到 Epic EMR，**"免费"** 给 Epic 客户

## 主流公司对比

## 典型使用流程

```
1. 医生进诊室，打开 Abridge / Nabla App
2. 医患对话（正常进行）
3. AI 实时转写 + 识别结构
4. 诊疗结束后 2 分钟：
   - 病历草稿
   - 医嘱草稿
   - ICD-10 编码建议
5. 医生审核 + 编辑（5 分钟）
6. 推入 EMR（Epic / Cerner）

节省时间：30-60 分钟/天
```

## 技术架构

- **ASR**（语音识别）+ LLM（GPT-4 / Claude 专用版）
- **医学实体识别 + ICD 编码**
- **EMR 集成**（Epic / Cerner / Meditech）
- **HIPAA 合规 + BAA 协议**

## 商业模式

- **Per-encounter**（每次就诊 $3-10）
- **Per-provider**（每医生 $300-500/月）
- **企业 合同**（$500k-10M ACV）

## ROI 清晰

- 节省医生时间：30-60 分钟/天
- 提高 patient throughput：+10-20%
- 减少 burnout + turnover
- **典型 payback 3-6 个月**

## 竞争动态

- **Abridge** 独立玩家领先
- **Microsoft DAX + Nuance** 分发最广
- **Epic Ambient** 免费绑定 Epic EMR
- **Nabla** 欧洲代表 + GDPR 友好

## 2026 关键变量

1. **Abridge 能否 IPO**（2026-2027 目标）
2. **Epic Ambient 免费策略**对独立玩家冲击
3. **中国临床 Copilot** 起步（ 讯飞医疗 / 商汤医疗）
4. **LLM + Multimodal** 扩展能力
