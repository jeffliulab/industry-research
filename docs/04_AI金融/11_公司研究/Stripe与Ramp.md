# Stripe · Ramp（B2B 金融 SaaS + AI）

> 最后更新：2026-04-24
>
> Stripe 和 Ramp 是 **美国 B2B 金融科技代表**——Stripe（2010 Collison 兄弟创立，$95B 估值）做 **支付基础设施**，Ramp（2019 创立，$16B 估值 2025）做 **企业支出管理 + AI 自动化**。两者都**深度集成 AI** 做自动对账、欺诈检测、发票识别、Agent 工作流。

## 一句话定位

两者 = **"美国 B2B 金融 AI 的两张脸"** —— **Stripe**（支付巨头 AI 化）+ **Ramp**（AI 原生 spend management）。**Ramp 更代表"AI 原生金融 SaaS"**，Stripe 代表"AI 赋能传统金融基础设施"。

## 一、两家速览

| 维度 | Stripe | Ramp |
|---|---|---|
| 成立 | 2010 | 2019 |
| 总部 | SF + Dublin | NYC |
| 创始人 | Patrick + John Collison | Eric Glyman + Karim Atiyeh |
| 估值 | **$95B**（2025 pre-IPO）| **$16B**（2025）|
| 员工 | ~10,000 | ~1,300 |
| 业务 | 支付基础设施 | 企业卡 + spend management |
| ARR | $14B+（2024）| $1B+（2025 新闻）|

## 二、AI 战略

### Stripe
- **Radar** 反欺诈（10 年 AI 积累）
- **Stripe Docs AI**（2024）：API 文档 AI 助手
- **Sigma Assistant** + **LLM 驱动 invoice 识别**
- 2024 -11 **OpenAI 合作** 让 ChatGPT 支付功能用 Stripe

### Ramp
- **AI-native spend management** 从创立第一天
- **Ramp Intelligence**（2024-11）：GPT-4 生成 **insight / anomaly detection**
- **自动分类 + 发票识别 + 供应商谈判 AI**
- 2024 **估值 $16B，ARR 增长 80%+**

## 三、业务与产品

### Stripe 产品
- **Payments**：支付
- **Connect**：平台
- **Billing**：订阅
- **Radar**：反欺诈
- **Stripe Apps**：集成 + AI

### Ramp 产品
- **企业卡**
- **Bill Pay**
- **Expense management**
- **Procurement**
- **Ramp Intelligence**（AI）

## 四、技术路线

- Stripe：自研 ML + OpenAI 合作
- Ramp：**AI-native 架构**，每个产品线集成 LLM

## 五、商业模式

- Stripe：2.9% + $0.30 per transaction
- Ramp：企业卡 interchange + SaaS 订阅

## 六、竞争与壁垒

### Stripe
- 对标 Adyen、Square Block、PayPal
- 壁垒：**Collison 兄弟 + 开发者 brand + API 体验**

### Ramp
- 对标 Brex、Concur（SAP）、Expensify
- 壁垒：**AI-native + CEO brand + 增长速度**

## 七、关键风险

- Stripe：IPO 定价、Square / Adyen 竞争
- Ramp：Brex 等对手、企业采购放缓

## 八、我的判断

> 1. **Stripe 是"AI 赋能" 代表**，Ramp 是"AI 原生" 代表
> 2. **Ramp 估值 $16B 增速惊人**，2026-2027 目标 $30B
> 3. **Stripe IPO 2026-2027 是关键事件**
> 4. **企业 AI 金融 SaaS** 是最清晰的 AI 落地赛道之一
>
> **我可能错在哪里**：Stripe 上市估值低于预期；Ramp 增长放缓。
