# Physical Intelligence (π)

> 最后更新：2026-04-23
>
> Physical Intelligence (pi / π) 是 2024-2025 年具身智能赛道**最受关注的"纯大脑"公司**。由 Karol Hausman、Sergey Levine、Chelsea Finn 等顶级研究者创立，目标是成为"**通用机器人大脑**"，不做整机。2025 年 Amazon 领投 Series C，估值跃升到 **~$5B**。

## 一、公司速览

- **创立**：2024 年 3 月
- **总部**：旧金山
- **创始人**：Karol Hausman、Sergey Levine、Chelsea Finn、Lachy Groom 等（UC Berkeley / Stanford 背景）
- **员工数**：~80+（2025 末，持续扩招）
- **估值**：**~$5B**（2025-11 Amazon 领投 Series C）
- **主要投资方**：Amazon、Thrive、Lux、Bezos Expeditions、OpenAI Startup Fund
- **一句话业务**：研发通用机器人基础模型（RFM），不做机器人硬件，目标成为"所有机器人的大脑供应商"

## 二、历史沿革

| 时间 | 事件 |
|---|---|
| 2024-03 | 创立 + $70M 种子 |
| 2024-10 | **π0 发布**：业界震动，技术报告引用大量 |
| 2025-02 | Series B（$400M） |
| 2025-10 | **π0.5 发布**：能力跃升，家庭长时程任务表现突出 |
| 2025-11 | Series C（Amazon 领投） |
| 2026-Q1-Q2 | **π1 预期**：下一代，路线图未完全公开 |

## 三、业务与产品

### π 系列模型
- **π0**（2024-10）：首款公开的大规模 VLA 模型
- **π0.5**（2025-10）：能力升级，家庭长时程突破
- **π1**（2026 预期）：下一代

### 合作机器人平台
π 的模型**不绑定单一硬件**，在多种机器人上运行：
- Franka 机械臂
- 开源双手机器人（ALOHA 系列）
- 未来可能扩展到人形

### 商业路径（预期）
- 向机器人整机厂授权 / 卖模型
- 直接向使用机器人的工业客户提供"机器人 + π 软件"打包方案
- **2025-2026 仍是研究为主，商业化初期**

## 四、技术路线

### 核心差异化
1. **VLA（Vision-Language-Action）端到端路线**（详见 [具身智能技术路线](../01_技术路线/技术路线.md)）
2. **真机数据优先**：10,000+ 小时遥操作真机数据积累
3. **跨形态设计**：π0 的 action tokenization 考虑跨机器人迁移
4. **学术背景深**：Sergey Levine 等是 RL / robotics 领域顶级学者

### 数据壁垒
- 10k+ 小时真机数据（2025 末）
- 数据是连续采集，随部署扩大指数增长
- 这是 Physical Intelligence 最强的 cornered resource

## 五、商业模式

### 当前（2024-2026 早期）
- 融资驱动（已融 $500M+）
- 少量研究合作收入

### 长期（2027+）
- **模型授权**：给 Figure / Boston Dynamics / 宇树等整机厂用 π
- **机器人解决方案**：与整机厂合作，提供完整 stack
- **可能自建机器人车队 / RaaS**：但暂无明确迹象

### 挑战
- 纯大脑公司的生态位困难——类比 Kiva（被 Amazon 收购）而不是独立存活
- **Amazon 战略投资**暗示"最终被 Amazon 吸收"的可能性

## 六、竞争与壁垒

### 直接对标
- **Skild AI**（CMU 系）：Skild Brain
- **Covariant**：工业场景 RFM-1（团队大部分已被 Amazon 吸收）
- **Google DeepMind RT / Gemini Robotics**：开放平台竞争
- **NVIDIA GR00T**：开放平台策略
- **整机厂自研**（Tesla / Figure / 1X）：上游竞争

### 用 [VRIO 框架](../../99_方法论/框架_VRIO.md)

| 资源 | V | R | I | O |
|---|---|---|---|---|
| 10k+ 小时真机数据 | ✅ | ✅ | ✅ | ✅ |
| 顶级学者团队（Levine / Finn / Hausman） | ✅ | ✅ | 🟡 | ✅ |
| π0 / π0.5 算法创新 | ✅ | ✅ | 🟡 | ✅ |
| Amazon 战略投资 + 合作 | ✅ | ✅ | ❌ | ✅ |

**最强 Power**：**数据 + 顶级团队**，两者都是 cornered resource。

## 七、关键风险

1. **生态位挑战**：纯大脑公司商业模式未经验证——整机厂最终会自研大脑
2. **Amazon 吸收可能性**：Amazon 战略投资 $5B → 历史上这类"战略投资 + 技术合作"常以被收购结束
3. **整机厂自研进度**：Tesla / Figure / 1X 都在自研大脑，可能不买 π
4. **开放平台竞争**：NVIDIA GR00T 免费给所有机器人厂，π 需要证明付费价值
5. **团队稳定**：学者创业公司人才吸引力大但也易被挖角

## 八、我的判断

> **我的看法**：
>
> 1. Physical Intelligence 是 **2024-2025 年具身智能技术最被看好**的公司之一——π0/π0.5 确实代表了技术最前沿
> 2. **商业模型存在结构性挑战**：纯大脑公司的生态位，历史上极少独立存活（Kiva、Covariant 都被 Amazon 吸收）
> 3. **2026-2028 π 面临关键选择**：做整机？被 Amazon 收购？保持独立纯大脑路线？这个决策决定长期命运
> 4. **我看好技术 + 团队**，但对"纯大脑独立走完全程"的**商业模式存疑**
>
> **我可能错在哪里**：
>
> - 如果 π1 继续数量级跃升，可能打破"纯大脑不存活"的历史规律
> - 机器人市场 fragmentation 很高 → 生态位充足，π 可以是"机器人界的 Android"
> - Amazon 的投资可能就是深合作不是收购前奏
