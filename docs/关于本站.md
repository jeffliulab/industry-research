# 关于本站

AI Industry Research 是一份聚焦 AI 及其相关产业的持续调研笔记，由 [@jeffliulab](https://github.com/jeffliulab) 维护。

## 立场与免责

- **不构成投资建议**：本站所有内容均为**公开信息整理**与个人观点记录，不构成任何投资、并购、求职决策建议
- **非内幕信息**：不发布任何未公开的公司内部信息
- **错误修正**：欢迎 issue / PR 指正事实错误或数据过期
- **Fair Use**：引用图表、数据时标明出处；如版权方认为构成侵权，可开 issue 要求下架
