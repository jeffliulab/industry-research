# Midjourney

> 最后更新：2026-04-24
>
> Midjourney 是 **2022 年 AI 图像生成最具辨识度的产品**——由前 Leap Motion 创始人 David Holz 2021 年创立于旧金山。**不融资、不上市、不接受风投**，靠纯订阅盈利，**2024 年 ARR $300-500M**，估值若按 20x 倍数计算 ~$6-10B，但 Holz 拒绝任何融资估值。Midjourney V6 / V7 图像质量至今仍是**艺术风格标杆**。

## 一、产品定位

Midjourney 是 **"AI 图像生成的艺术派" + "用户社区最强的 AI 产品"** —— 独特之处：
- **全部用 Discord 接入**（2022-2024，后加 Web）
- **不融资 + 盈利的独角兽**（业界罕见）
- **艺术风格强烈**：美学 + 画面感远胜 DALL-E / SD
- **社区强**：Discord 数百万活跃用户互相观摩

## 二、核心能力与架构

### 核心能力
- **文字 → 图像**（核心）
- **图像 → 图像**（参考风格 / 混图）
- **Describe**：图像 → 描述 prompt
- **Zoom / Pan**：画布外扩
- **Vary Region**：局部修改
- **Style References / Character References**：一致性参考
- **Midjourney V7**（2025）+ Video Beta（2025 Q4）

### 技术架构
- 自研 Diffusion 模型（V1 → V7）
- 不基于 Stable Diffusion（早期有关但分叉）
- 训练数据：自有许可 + 公开
- 算力：自建 + 云租赁

### 规格（V7）
- 最高分辨率：2048×2048
- Prompt：自然语言 + 技术参数（--ar --stylize --chaos）
- 支持视频 Beta（2025 末）

## 四、定价与商业化

### 订阅
| 层级 | 月费 | 额度 |
|---|---|---|
| **Basic** | $10 | 200 张快速 |
| **Standard** | $30 | 15 小时快速 + 无限慢 |
| **Pro** | $60 | 30 小时快速 + Stealth |
| **Mega** | $120 | 60 小时快速 |

### 商业模式
- **纯订阅** —— 无广告、无企业版、无API
- **创始人拒绝融资**
- 2024 估算 ARR **$300-500M**
- **员工 40 人**（业界最小规模大公司）

### 盈利能力
- **毛利 60%+**（图像生成成本不高）
- **LTV 长**（用户粘性强）
- **员工少** → 成本低
- 估算净利 $100M+

## 五、用户反馈

### 艺术家社区
- **"审美最佳的 AI 图像工具"**
- **Discord 社区文化独特**：prompt 公开可见 + 互相启发
- **独立插画师 / 设计师** 大量使用

### 普通用户
- **"学习曲线比 DALL-E 陡"**：Discord 命令 + 参数
- **Web 版本后** 用户体验大幅改善
- **艺术质感** 是普通用户首选

### 批评
- **版权诉讼**：Hollywood / 插画师集体起诉（2024-2025）
- **无 API** 限制企业集成
- **V6 起收紧不雅 / 政治内容** 引发部分用户反弹
- **V7 发布 不如 V6 → V5 那么惊艳**

## 七、使用笔记

### 最适合
1. **艺术插画 / 海报 / 概念设计**
2. **电商图 / 广告背景**
3. **社交内容 / 博客配图**
4. **创意探索**（风格试错）

### 不太适合
- **需要精确文字 / 可读文字**（Flux / DALL-E 更准）
- **企业 API 集成**（没 API）
- **精确人物一致性**（Style/Character Ref 有限）
- **极速生成**（相对慢）

### 工作流典型
```
/imagine a vibrant cyberpunk city at night,
  neon reflections, rain, 35mm film, ultra detailed
  --ar 16:9 --stylize 750 --v 7

→ 4 张候选
→ U2 放大第二张
→ Vary Region 修改局部
→ Zoom Out 扩展画面
```
