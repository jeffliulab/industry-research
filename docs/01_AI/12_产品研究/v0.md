# v0（Vercel）

> 最后更新：2026-04-24
>
> v0 是 **Vercel 2023-10 推出的 AI UI 生成工具** —— 输入自然语言或截图，生成可直接部署的 React / Next.js / Tailwind 前端代码。定位介于**"设计 → 代码"和"产品 → 部署"之间**，作为 Vercel 生态的 Agent-first 产品线。2025 成为 Vercel 用户增长主要引擎。

## 一、产品定位

v0 是 **"AI 原生 UI Builder + 快速原型工具"**——适合 **产品经理 / 设计师 / 创业者 / 前端工程师** 快速从"想法 / 设计" 变成"可部署 UI"。与 Bolt / Lovable / Replit 同属"Low-Code AI Coder"赛道，但 **集成 Vercel 部署 + shadcn/ui 生态** 是独特优势。

## 二、核心能力与架构

### 核心能力
- **文字生成 UI**：自然语言 → React 组件
- **图像 → UI**：截图 / Figma → 代码
- **shadcn/ui 原生**：Radix + Tailwind 组件库
- **多文件项目**：不仅单组件，全 Next.js 应用
- **一键部署 Vercel**：从生成到上线
- **v0 Chat**：迭代式对话修改

### 底层
- Claude 3.5 / 4 Sonnet 主力
- GPT-4o 备选
- fine-tune 在 React / Tailwind / shadcn 语料
- Vercel 自家 infrastructure

## 五、用户反馈

### 正面
- **"最快的原型工具"** —— 推特常见评价
- **Figma → 代码**的 gap 被有效缩小
- **shadcn/ui 质量高**：生成的 UI 好看可用

### 批评
- **仅前端能力**（2025 起加 Backend）
- **生成代码有时冗余**
- **不如手写精细**
- **价格偏贵**（相比 Bolt / Lovable 免费额度）

### 与 Bolt / Lovable / Replit 对比
- **v0**：设计师 / PM / 前端工程师
- **Bolt.new**：非开发者 / 创业者（StackBlitz 后端）
- **Lovable**：快速原型 → 上线
- **Replit**：全栈 + 部署一体化

## 七、使用笔记

### 最适合
1. **产品经理快速验证 UI 想法**
2. **设计师 → 代码"最后一公里"**
3. **创业者做 MVP**
4. **已用 Vercel + Next.js 的团队**
5. **Dashboard / 落地页 / 内部工具**

### 不太适合
- **复杂业务逻辑后端**：Replit / 手写更好
- **大型企业级应用**：生成代码需大量重构
- **成本敏感用户**：$20 起门槛

### 工作流典型
```
v0 prompt: "a SaaS pricing page with 3 tiers,
  Tailwind + shadcn/ui, dark mode toggle"
→ 生成代码 + 预览
→ Chat: "make the middle tier highlighted,
   add annual/monthly toggle"
→ 再生成
→ "Deploy to Vercel" 一键
```
