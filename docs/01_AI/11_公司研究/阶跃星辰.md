# 阶跃星辰（StepFun）

> 最后更新：2026-04-24
>
> 阶跃星辰是**六小虎中最"AGI 信仰派"的玩家**——2023-04 姜大昕（前微软亚研院 / 上海 AI 实验室）创立，专注**多模态 + 推理 + 具身智能**的长远 AGI 路径。估值 $2B，虽然流量产品弱，但技术储备 Step-2 / Step-3 系列在多模态评测中表现出色。

## 一句话定位

阶跃星辰是 **"中国的 Anthropic"**——不追 C 端流量，不做大量垂直，而是**专注多模态 + 长期 AGI 研究 + 特定 B 端定制（智能汽车、终端设备）**。2025 年估值 $2B，规模在六小虎中偏小但**技术存活能力最强**之一。

## 一、公司速览

- **创立**：2023-04
- **总部**：上海
- **创始人 / CEO**：**姜大昕**（前微软亚研院首席研究员，上海 AI Lab 主任）
- **员工**：~200（2025 末）
- **估值**：**$2B**（2024-09，B 轮 $300M+）
- **主要投资方**：上海国资、红杉中国、启明创投、真格基金、腾讯

## 三、业务与产品

### 模型矩阵
| 模型 | 定位 |
|---|---|
| **Step-2** | 千亿 MoE 旗舰 |
| Step-1 / Step-1V | 通用 + 视觉 |
| **Step-Video** | 视频理解 |
| **Step-Audio** | 语音 |
| **Step-R** | 推理模型（对标 o1 / R1）|
| **Step-3** | 2025-Q4 新版本 |

### 产品
- **跃问 App**：C 端对话（低调，MAU 百万级）
- **阶跃 API**：企业版
- **Step-Omni**：多模态全栈

### 目标客户
- **智能汽车**：与某头部车企深度合作
- **具身 / 机器人**：智元等机器人公司
- **企业垂直 API**：金融 / 医疗少数定制

## 四、技术路线

### 多模态优先
- **Step-Video**：视频理解 + 生成融合
- **Step-Audio**：对话 + 语音合成一体
- **Step-1V / Step-2V**：视觉 SOTA
- **原生多模态训练**（非 bolt-on）

### 推理 + AGI 路径
- **Step-R** 推理模型路线
- 相信 "System 2 思考"是通往 AGI 关键
- 长远研究 > 短期产品

### 技术团队
- **姜大昕个人 brand**：微软亚研院首席 + 上海 AI Lab 主任
- 团队学术能力强，与学界合作紧密

### 算力
- 上海 AI Lab 合作（政府算力）
- 自建 + 租用火山引擎 / 阿里云

## 五、商业模式

### 收入来源
| 来源 | 2025 估算 |
|---|---|
| 企业 API | ¥50M |
| 智能汽车 / 机器人定制 | ¥100M |
| 政府项目（上海 AI Lab 合作）| ¥50M |
| **2025 总** | **~¥200M（$27M）** |

### 挑战
- C 端产品弱，品牌认知不足
- ARR 规模小
- 估值 $2B / $27M ARR = 74x（高但不离谱）

## 六、竞争与壁垒

### 对标
- **DeepSeek**：多模态上 DeepSeek 路线类似但 DeepSeek 更聚焦通用
- **智谱**：同为学术派
- **Moonshot**：阶跃没有 Kimi 级 C 端但技术更全面

### 壁垒
- **多模态技术储备**：Step-Video / Audio / Image 在行业内领先
- **姜大昕 brand**：微软亚研院 + 上海 AI Lab
- **上海政府支持**：基础设施 + 资金保障
- **车企 / 具身长期绑定**

### 弱点
- **C 端品牌近乎零**：跃问无人知晓
- **商业化节奏慢**：工程落地 vs 研究偏慢
- **规模小**：团队只有 200 人，很难与阿里 / 字节规模竞争

## 七、关键风险

- **资金消耗**：多模态训练成本高
- **被大厂挖角**：上海 AI Lab 背景员工易流失
- **具身 / 车企合作集中度高**：若某个客户变化，收入受冲击
- **姜大昕个人重要性**：技术灵魂人物
- **政府关系变化**：上海地方政府支持可能不如预期

## 八、我的判断

> **我的看法**：
>
> 1. **阶跃星辰是六小虎中**最可能做出技术突破**的玩家**——多模态 + 推理双线储备扎实
> 2. **商业化会是最慢的**：姜大昕更像研究者，不是商业大师
> 3. **终局**：**被大厂收购**（腾讯概率大）或**成为上海 AI 国家队成员**
> 4. **多模态技术储备有真实价值**：2026 若 Sora 级视频生成开源，阶跃有望成为中国头部
> 5. **具身智能 / 机器人合作** 是重要押注——若机器人行业爆发，阶跃技术赋能价值大
>
> **我可能错在哪里**：
> - 阶跃做出 Sora / Gemini 级视频 / 多模态模型突破，估值飞升
> - 跃问产品突然爆火（需要大补贴）
> - 被腾讯以 $5B+ 整体收购
