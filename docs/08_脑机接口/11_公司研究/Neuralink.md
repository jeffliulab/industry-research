# Neuralink

> 最后更新：2026-04-24
>
> Neuralink 是 **Elon Musk 2016 创立的侵入式 BCI 公司**——**2024-01 首例人体植入**（Noland Arbaugh 高位截瘫患者），震惊全球。**2024-05 第二例**，**2025 多例**。**N1 Chip + R1 手术机器人 + Blindsight（视觉假体）** 是核心产品。估值 ~$15B（2025）。

## 一句话定位

Neuralink = **"BCI 行业的 SpaceX / Tesla"** —— Elon 品牌 + 激进研发 + 硬件工程结合。**让 BCI 进入大众视野**。**医疗应用先行**（瘫痪），**消费版承诺 2030+**。

## 一、公司速览

- **创立**：2016
- **总部**：Austin + Fremont
- **CEO**：Elon Musk
- **员工**：~500
- **估值**：**$15B**（2025 推测）
- **投资方**：Founders Fund、Google Ventures、Elon 个人

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2016 | 成立 |
| 2019 | 首次公开 Demo |
| 2020 | 猪 Demo |
| 2021 | 猴子玩 Pong |
| 2023-05 | **FDA 批准人体试验**（BREACH） |
| **2024-01** | **首例人体植入**（Noland Arbaugh）|
| 2024-05 | 第二例 |
| 2024-09 | **Blindsight FDA Breakthrough**（视觉假体）|
| 2025 | 累计 10+ 人体试验目标 |

## 三、业务与产品

- **N1 Chip**：1024 通道神经信号
- **R1 Robot**：手术机器人（植入 Thread）
- **Thread**：柔性电极线
- **Blindsight**：视觉假体 （研发中）

## 四、Noland Arbaugh 首例情况

- 2024-01 植入
- 2 个月时 Thread 部分 retracted（脱落）
- Noland 仍能用 App 控制鼠标
- **"最大成就" + "最大教训" 双面**

## 五、技术路线

- **皮层穿透 Thread**（柔性）
- **R1 手术机器人自动植入**
- **N1 无线 + 电池**
- **AI 解码**：自研

## 六、商业模式

- 未商业化
- 医疗 BCI 批准后 per-procedure 收费（估 $50-200k）
- 长期消费版未知

## 七、竞争与壁垒

- 对标：Synchron、Paradromics、Precision、BrainGate
- 壁垒：**Elon 资金 + 品牌 + 工程规模**
- 弱点：**侵入式风险** + **Noland 案例显 Thread 稳定性问题**

## 八、关键风险

- **人体试验重大事故**
- **FDA 加严监管**
- **Elon 分心**（Tesla / SpaceX / xAI / 政府）
