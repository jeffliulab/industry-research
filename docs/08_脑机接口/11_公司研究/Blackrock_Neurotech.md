# Blackrock Neurotech

> 最后更新：2026-04-24
>
> Blackrock Neurotech 是 **2008 年 Florian Solzbacher + Marcus Gerhardt 创立的老牌 BCI 硬件公司**（犹他州盐湖城）——**生产 Utah Array 电极**，是 **BrainGate 和大多数学术 BCI 研究的电极供应商**。**2024 年获 $200M 投资**（Tether / iFinance 主导），**Utah Array 2.0 + NeuroPort 新产品**。

## 一句话定位

Blackrock = **"BCI 电极行业 ARM"** —— 不是明星消费品牌，**但大部分人体 BCI 植入都用它的电极**。**Utah Array 几乎是学术 BCI 事实标准**。**2024 资金注入后加速商业化产品**。

## 一、公司速览

- **创立**：2008
- **总部**：Salt Lake City, Utah
- **创始人**：Florian Solzbacher
- **员工**：~150
- **估值**：未披露（2024 $200M funding）
- **主要投资方**：Tether（母公司 iFinance）

## 二、产品

- **Utah Array 64 / 96 通道**：学术金标准
- **MoveAgain BCI System**：FDA Breakthrough 2022
- **NeuroPort**：信号放大 + 数字化
- **Cortical Visual Prosthesis**：视觉假体

## 三、历史

- **1990s Utah Array 诞生**（犹他大学）
- 2008 Blackrock 成立商业化
- **BrainGate 25 年累计 15+ 例患者** 用 Utah Array
- 2022 **首个 "BCI" 名称 FDA Breakthrough**

## 四、商业模式

- 学术电极销售（数千美元 / 单位）
- 医疗系统 FDA 批准后 $$
- 2024 资金注入后扩大商用
