# BrainGate（学术联盟）

> 最后更新：2026-04-24
>
> BrainGate 不是单一公司，而是 **2002 年起的学术联盟**——布朗大学 + 斯坦福 + 麻省总医院 + Case Western 等。**使用 Utah Array 植入瘫痪患者** 让其控制光标 / 机械手。**2004 首例人体植入**（Matt Nagle），**至今累计 15+ 例患者**。**是 BCI 历史最悠久的人体试验项目**。

## 一句话定位

BrainGate = **"BCI 学术金标准"** —— 20 年人体试验经验 + 论文 + Utah Array 协同。**不是 Neuralink 式公司化**，**是大学联盟 + NIH 资助**。**许多 BCI 创业者都来自 BrainGate 研究**。

## 一、联盟速览

- **成立**：2002
- **主要机构**：
  - 布朗大学（Leigh Hochberg）
  - 斯坦福（Krishna Shenoy，已故）+ Jaimie Henderson
  - 麻省总医院
  - Case Western
- **资助**：NIH、NSF、VA
- **企业合作**：Blackrock Neurotech（电极供应）

## 二、历史里程碑

| 时间 | 里程碑 |
|---|---|
| **2004** | **Matt Nagle 首例人体植入** |
| 2012 | Cathy Hutchinson 用机械臂喝咖啡 |
| 2017 | 闭环 BCI 打字 |
| 2021 | **Dennis DeGray 语音 BCI** |
| 2023 | **Pat Bennett 语音 BCI** 62 words/min |

## 三、核心贡献

- **Utah Array 电极** 的临床验证
- **光标 / 键盘 / 机械臂 / 语音** 解码算法
- **长期安全性** 数据（10+ 年）
- **训练下一代 BCI 科研人员**

## 四、与企业的关系

- **Blackrock Neurotech**（电极硬件）商业化
- **Neuralink 团队部分来自 BrainGate 训练**
- **Synchron CEO Tom Oxley 学术出身**

## 五、挑战

- **学术联盟资金**：与 Neuralink 数十亿美元比相对少
- **商业化慢**：必须通过 FDA 等正式路径
- **规模小**：每年植入 1-2 例

## 六、我的判断

> 1. **BrainGate 是 BCI 学术基石** —— 没有它就没有 Neuralink
> 2. **未来更多作为研究平台**，不是商业化主力
> 3. **多数 BCI 创业公司都受 BrainGate 训练**
>
> **我可能错在哪里**：BrainGate 商业化分拆成新公司，资金 + 规模大增。
