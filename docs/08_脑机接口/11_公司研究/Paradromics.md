# Paradromics

> 最后更新：2026-04-24
>
> Paradromics 是 **2015 创立的高带宽 BCI 公司**——Matt Angle 创立，**Connexus 系统 1600+ 通道**（比 Utah Array 高 10x）。**2023 FDA Breakthrough Device**，**2024 人体试验准备**。**通道数最大的 BCI 系统**。

Paradromics = **"通道密度最高的 BCI"** —— 与 Neuralink Thread 竞争"**下一代高带宽 BCI**"。**Matt Angle 学术背景** + **专注数据量而非快速试验**。
