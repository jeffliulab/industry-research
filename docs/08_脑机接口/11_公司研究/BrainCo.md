# BrainCo 强脑科技

> 最后更新：2026-04-24
>
> BrainCo 是 **2015 Harvard Innovation Lab 出身的消费级 BCI 公司**——韩璧丞创立。**Focus 1 EEG 头带** 在中国教育场景销量 200 万+。**2024 上市前 pre-IPO 估值 ~$1B**。**也做智能假肢**（BrainRobotics）和**康复**（BrainMatrix）。

## 一句话定位

BrainCo = **"中国消费 BCI 规模最大"** —— 教育 / 注意力 + 智能假肢 + 康复 三线产品，**在中国 B 端 + C 端销售**。**2024 争议：教育监控伦理**让 BrainCo 面临社会批评。

## 一、公司速览

- **创立**：2015（Harvard Innovation Lab）
- **总部**：深圳 + Boston
- **创始人 / CEO**：韩璧丞
- **员工**：~500
- **估值**：~$1B（2024 pre-IPO）

## 二、产品

- **Focus 1 EEG 头带**：注意力训练
- **BrainRobotics 智能假肢**
- **BrainMatrix 康复训练**
- **Mindfulness App**

## 三、销售规模

- **Focus 1**：200 万 + 销量（2024 累计）
- 主要中国 **学校 + 教育机构**
- 出海 美国 + 东南亚

## 四、技术路线

- EEG（头皮电极）
- **轻量 + 便携**（消费友好）
- AI 解码（学生注意力）

## 五、争议

### 2019 教育监控事件
- **浙江金华某学校 让学生戴 Focus 1** 监控注意力
- 社会批评 隐私 + 心理伤害
- BrainCo 回应：**强调自愿 + 数据不存**

## 六、商业模式

- 硬件销售（$200-500）
- **教育 B2B 渠道** 为主
- 假肢 / 康复 医疗合作

## 七、竞争与壁垒

- 对标：Muse、Emotiv、Kernel
- 壁垒：**中国教育渠道 + Harvard brand + 多产品线**
- 弱点：**消费 BCI 通用天花板 + 监控伦理争议**

## 八、我的判断

> 1. **BrainCo 是消费 BCI 规模最大**（主要 Focus 1 销售）
> 2. **教育监控争议** 是长期负担
> 3. **智能假肢是更好故事线**
> 4. **IPO 目标 2026-2027**
>
> **我可能错在哪里**：教育禁用 BrainCo 商品导致增长停滞。
