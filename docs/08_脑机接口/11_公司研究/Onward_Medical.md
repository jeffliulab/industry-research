# Onward Medical

> 最后更新：2026-04-24
>
> Onward Medical 是 **荷兰 2014 创立的脊髓损伤 BCI + 神经刺激公司**——**2024 与瑞士 EPFL 合作 BCI + 脊髓刺激**让截瘫患者**重新行走**（Gert-Jan Oskam 案例）。**2021 Euronext 上市**。**专注脊髓损伤而非皮层 BCI**。

## 一句话定位

Onward = **"脊髓 BCI + 神经刺激 独特路径"** —— 不做 Neuralink / Synchron 路径，**组合 脑电极（读） + 脊髓刺激（写）让 瘫痪者行走**。**是 BCI 临床最戏剧化案例之一**。

## 一、公司速览

- **创立**：2014
- **总部**：Eindhoven, NL + Lausanne
- **CEO**：Dave Marver
- **员工**：~70
- **上市**：**Euronext ONWD**（2021）
- **市值**：~€200M

## 二、合作 / 里程碑

- **2023 Gert-Jan Oskam 案例**（Nature）：瘫痪 12 年患者通过 BCI + 脊髓刺激重新行走
- **与 EPFL / 瑞士联邦理工** 长期合作
- **FDA Breakthrough Device**

## 三、产品

- **ARC-BCI**：数字桥接（脑 + 脊髓）
- **ARC-EX**：非侵入神经刺激
- **ARC-IM**：植入式

## 四、技术路线

- **脑电极 → WiMagine → 脊髓刺激**
- **AI 解码脑信号 → 触发脊髓刺激**
- 与 Utah Array 或 Synchron 等互补

## 五、商业模式

- 医疗设备销售
- **2024-2025 临床推进**
- 2024 营收 ~€10M

## 六、我的判断

> 1. **Onward 是独特路径 + 戏剧性案例**
> 2. **脊髓损伤 目标市场小但明确**

> 4. **被 Medtronic / Boston Scientific 收购** 可能
