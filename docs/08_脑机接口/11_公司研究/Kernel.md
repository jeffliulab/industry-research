# Kernel

> 最后更新：2026-04-24
>
> Kernel 是 **Braintree / Venmo 创始人 Bryan Johnson 2016 创立的非侵入式 BCI 公司**——专攻 **fNIRS（功能近红外）**。旗舰产品 **Flux**（头戴设备）。2020 **$100M 融资**后商业化进度缓慢。**Bryan Johnson 2023 起转型"反衰老"**（Blueprint），Kernel 业务萎缩。

## 一句话定位

Kernel = **"消费 BCI 最大烧钱案例"** —— Bryan Johnson 个人投入 $50M+ + 外部 $100M+，但**fNIRS 消费化未成功**。**2023-2025 公司渐隐**，**Bryan 转型 Blueprint 反衰老更火**。

## 一、公司速览

- **创立**：2016
- **总部**：Culver City, CA
- **CEO**：Bryan Johnson（转型后退居二线）
- **员工**：~30（峰值 70）
- **累计融资**：$130M+
- **估值**：未披露 新一轮（2025）

## 二、产品

- **Kernel Flux**：fNIRS 头戴
- **Kernel Flow**：更早版本
- **价格 $50k 起**（科研用）

## 三、历史

| 时间 | 里程碑 |
|---|---|
| 2016 | 成立 |
| 2020 | B 轮 $100M |
| 2021 | Flux / Flow 对外 |
| 2023 | 商业化艰难 |
| 2024-2025 | **规模缩减 + Bryan 专注 Blueprint** |

## 四、技术路线

- **fNIRS**（近红外血氧）
- 比 EEG 信号好 + 比侵入安全
- 但**产品成本高 + 用例不清**

## 五、商业模式困境

- **科研市场太小**
- **消费者不愿 $50k**
- **2024 多个合作项目结束**

## 六、我的判断

> 1. **Kernel 是"非侵入式 BCI 商业化"失败代表**
> 2. **Bryan Johnson 2024 更多关注 Blueprint** 让 Kernel 式微
> 3. **fNIRS 技术本身有价值** 但产品 / 市场匹配失败
>
> **我可能错在哪里**：fNIRS 技术某个消费爆款出现，Kernel 起死回生。
