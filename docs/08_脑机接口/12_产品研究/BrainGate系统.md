# BrainGate 系统

> 最后更新：2026-04-24
>
> BrainGate 是 **BrainGate 学术联盟的核心 BCI 系统**——**Utah Array 电极 + BrainGate 解码软件 + 外部设备**。**20 年人体试验积累**（2004 至今）+ **15+ 例患者数据**。**不是商业产品，是学术研究平台**。**联盟详见 [BrainGate 公司](../11_公司研究/BrainGate.md)**。

## 一、系统组成

- **Utah Array 96 通道电极**（Blackrock 生产）
- **BrainGate Decoder 软件**
- **外部计算机 + 显示**
- **机械手 / 机械臂**（部分案例）

## 二、典型用例

- 光标控制
- 打字
- 机械手 / 机械臂
- **2023 语音 BCI**（Pat Bennett 62 words/min）

## 三、关键患者案例

| 案例 | 时间 | 功能 |
|---|---|---|
| **Matt Nagle** | 2004 | 首例光标控制 |
| **Cathy Hutchinson** | 2012 | 机械臂喝咖啡 |
| **Dennis DeGray** | 2021 | 意念打字 |
| **Pat Bennett** | 2023 | **语音 BCI** 62 w/m |

## 四、商业化

- **不商业化**（学术联盟）
- **数据 + 经验**输出给新创公司
- **Blackrock Neurotech**（电极供应商）商业化 Utah Array

## 五、对行业影响

- **Utah Array** 仍是 2026 学术 BCI 金标准
- **20 年数据** 是 FDA 审批依据
- **训练 Neuralink / Synchron 创始人们** 的前辈
