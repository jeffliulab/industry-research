# Neuralink N1 植入

> 最后更新：2026-04-24
>
> Neuralink N1 是 **Neuralink 的第一代人体 BCI 系统**——**1024 通道柔性 Thread** + **N1 Chip**（无线 + 电池）+ **R1 手术机器人**。**2024-01 首例植入 Noland Arbaugh**，**2024-05 第二例**，2025 累计 ~10 例。**公司详见 [Neuralink](../11_公司研究/Neuralink.md)**。

## 一、产品规格

- **N1 Chip**：无线 + 电池
- **Thread**：1024 电极 × 64 柔性线
- **R1 Robot**：手术机器人植入
- **App / SDK**：用户端控制

## 二、植入过程

1. 麻醉后开 ~1cm² 颅孔
2. R1 机器人自动植入 64 根 Thread
3. 闭合
4. N1 Chip 外接 / 无线充电
5. 康复 + 训练 AI 解码

## 三、首例案例（Noland Arbaugh）

- **2024-01 植入**
- 2 个月时 **Thread 部分脱落**
- 仍能用 App 控制鼠标 + 玩游戏
- **"最大成就" + "最大教训"**

## 四、第二例（2024-05）

- 改进植入技术
- Thread 稳定

## 五、后续（2025）

- 多例人体试验
- 数据累积
- 稳定性 + 可靠性改进

## 六、竞品对比

| 维度 | Neuralink N1 | Synchron Stentrode | Precision NSX | Utah Array |
|---|---|---|---|---|
| 植入方式 | 开颅 + Thread | 血管 | 薄膜贴表 | 穿透 pin |
| 通道 | 1024 | ~16 | 1024/片 | 96 |
| 侵入性 | 高 | 低 | 中 | 高 |
| 2025 人体数 | ~10 | ~10 | ~30 | 15+ |
