# Adept（历史 · 被 Amazon 技术性收购）

> 最后更新：2026-04-24
>
> Adept 是 **Agent 时代最早最被看好的公司之一**——2022 年由前 OpenAI / Google / DeepMind 研究员 **David Luan**（OpenAI 前应用 VP）、**Niki Parmar**（Transformer 论文合著者）、**Ashish Vaswani**（Transformer 论文一作）共同创立。2023 估值 $1B，**2024-06 被 Amazon 以 "reverse acqui-hire" 收购**（创始人 + 团队加入 Amazon AGI Lab），**是 Inflection / Character.AI / Windsurf 之前的"Agent 技术性收购"先例**。

## 一句话定位

Adept 是 **"Agent 先驱 + 商业化失败 + 被大厂人才收割"的典型案例** —— 技术团队顶尖（Transformer 创造者们！），但未能把"操作电脑"的 demo 变成商业化产品，最终 2024-06 被 Amazon 技术性并购。成为**"Agent wrapper 公司命运"** 的警示教材。

## 一、公司速览（2024 并购前）

- **创立**：2022-04
- **总部**：旧金山
- **创始人**：
  - **David Luan**（CEO，前 OpenAI 应用 VP、Google Brain）
  - **Niki Parmar**（前 Google Brain，Transformer 合著者）
  - **Ashish Vaswani**（前 Google Brain，Transformer **一作**）
  - Kelsey Szot、Augustus Odena、Maxwell Nye、Erich Elsen 等
- **员工**（峰值）：~40-50
- **2023 估值**：$1B（B 轮 $350M，2023-03）
- **投资方**：Greylock、Spark Capital、Addition、General Catalyst、NVIDIA、Microsoft、Atlassian、Workday、SAP

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2022-04 | Adept 成立 |
| 2022-04 | A 轮 $65M |
| 2022-09 | **ACT-1 发布**：早期 Agent，操作浏览器做任务（惊艳 demo）|
| 2023-03 | B 轮 $350M，估值 $1B（独角兽）|
| 2023-10 | **ACT-2 / Fuyu-8B** 发布（多模态 Agent 模型）|
| 2024-01 | Niki Parmar + Ashish Vaswani **离开 Adept** 创立 Essential AI（Transformer 原作者再出发）|
| 2024-06 | **Amazon "技术性收购"**：David Luan + 核心团队加入 Amazon AGI SF Lab |
| 2024-06 | Adept 作为公司继续存在（licensing）但失去核心 |
| 2025 | 剩余团队做 "Adept API"（Agent 企业服务），影响力极小 |

## 三、核心技术与产品

### ACT-1（Action Transformer）
- 2022 最早的**浏览器操作 Agent demo**
- Demo 视频：Agent 在 Salesforce / Redfin 里填表 / 搜索
- 引爆业界"Agent 革命" 期待

### ACT-2 + Fuyu-8B（2023）
- 多模态 Agent 基础模型
- Fuyu-8B 小而强（vision + language 一体）
- 开源部分组件

### 商业化尝试
- **Adept for Work**：企业 B 端 Agent
- 2023 几家大企业试用（包括 Atlassian、Workday 的战略投资）
- **未能规模化收入**

## 四、为什么失败

### 原因 1 · 基础模型时代不对齐
- 2022-2023 Adept 做自研模型 + Agent wrapper
- **GPT-4 / Claude 基础模型能力追平并超越 Adept 自研**
- Agent wrapper 差异化消失

### 原因 2 · 商业化路径不清
- B 端客户试用多但付费少
- 没有清晰 killer app
- Sierra / Harvey / Claude Code 都有更明确定位

### 原因 3 · 烧钱快
- $415M 累计融资快速消耗
- 2024 初已有现金流压力

### 原因 4 · 创始人分裂
- Transformer 一作 Ashish Vaswani / Niki Parmar 2024 离开
- 创始团队完整性不在
- 信号：内部路线分歧

### 原因 5 · 投资人压力变现
- 2024 AI 估值开始理性
- 投资人宁接受 **Amazon 收购创始人 + 团队** 也不愿继续烧钱

## 五、Amazon "技术性收购"的机制

- Amazon **不收购 Adept 公司本身**（避免反垄断审查）
- 但**招聘 David Luan + 核心团队加入 Amazon AGI SF Lab**
- **支付 Adept 公司"授权费"**（估算 $300-400M）获得 IP 使用权
- Adept 作为空壳继续运营（类似 Inflection 模式）
- **投资人拿回 1-1.5x 本金**（远低于 unicorn 预期）

## 六、遗产与影响

### 对 Agent 行业的影响
- **ACT-1 Demo 启发了整个 Agent 浪潮**
- David Luan、Niki Parmar、Ashish Vaswani 是 Agent 思想的重要贡献者
- Adept 的失败**验证了"模型公司不是 wrapper"** 路径难走

### 对投资人的警示
- Agent wrapper 公司的高风险
- 基础模型能力追赶时差异化消失
- "技术性收购"是新型"失败模式"

### 对大厂的影响
- Amazon AGI Lab 获得顶级人才
- 对 AWS Agent / Bedrock Agents 产生直接贡献

## 七、2024-06 之后 Adept

- 空壳公司
- 剩余 ~10-20 员工
- API 业务影响力极小
- **2026 初 Adept 品牌几乎消失**

## 八、我的判断

> **我的看法**：
>
> 1. **Adept 是 AI 史上最被高估后又被最遗忘的公司**：巨星级创始人 + $1B 估值 + 无商业化
> 2. **"技术性收购"是 Agent wrapper 的可预见结局**：Inflection / Character / Adept / Windsurf 都是这条路
> 3. **Transformer 创造者也做不成 Agent 公司** 证明：**技术 ≠ 商业**
> 4. **David Luan 在 Amazon 可能做出更大影响**：AGI Lab 资源 + Amazon Bedrock 落地能力
> 5. **Adept 的警示对今天 Cursor / Cognition / Sierra 都适用**：**何时变现、怎么变现 > 技术领先**
>
> **我可能错在哪里**：
> - Adept "空壳" 2026 起复活（概率低）
> - Amazon AGI Lab 成就未来"第二个 OpenAI"
> - Transformer 原作者们在 Essential AI 取得惊人突破
