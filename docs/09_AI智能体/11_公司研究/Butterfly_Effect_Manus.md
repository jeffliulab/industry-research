# Butterfly Effect（Manus 背后公司）

> 最后更新：2026-04-24
>
> Butterfly Effect（中文"蝴蝶效应"）是 **中国 AI Agent 赛道最大的爆款公司**——2025-03 发布 **Manus**（英文名同）后**震惊全球**：一款"通用 AI Agent" 产品在全球引爆，申请 beta 的等待列表百万级，被称作"**中国 OpenAI Operator**"或"**中国 Devin**"。创始人**肖弘**（Yichao Ji，毕业于华中科技大学）此前做过工具类产品。2025-10 融资 $75M，估值 **$500M+**。

## 一句话定位

Butterfly Effect（Manus）是 **"2025 年中国 AI 最大黑马"**——做通用 Agent（不限于 Coding），2025-03 发布首日即获得全球关注，**但受质疑"套壳 Claude"**（Agent 框架 + Claude 模型）。**是中国 AI 应用层难得的全球关注产品**。

## 一、公司速览

- **创立**：2022（前身为"Monica AI 助手"）
- **总部**：北京 / 新加坡（双总部）
- **创始人 / CEO**：**肖弘**（Yichao Ji）
- **员工**：~150（2025 末）
- **估值**：**~$500M+**（2025-10，B 轮 $75M）
- **主要投资方**：红杉中国、腾讯、Benchmark（美）、ZhenFund、HSG

## 三、业务与产品

### Manus（通用 Agent）
- **形态**：Web 应用，用户输入自然语言任务
- **能力**：
  - 研究 / 报告撰写
  - 代码编写 + 运行
  - 数据分析 + 可视化
  - 浏览器操作
  - 文件处理（PDF / Excel）
- **时长**：5 分钟到数小时
- **输出**：结构化报告 + 附件

### Monica（旧产品）
- **Chrome 浏览器 AI 助手**
- 已被 Manus 主流替代
- MAU 仍有 ~几百万

### 代表用例
- "帮我写一份中国锂电池产业 2025 年研报"
- "帮我做一个网站 → 部署到服务器"
- "分析这个 CSV 并画图"
- "订一张明天从北京到东京的机票"

## 四、技术路线

### 底层模型
- **主要调用 Claude 3.5 / 4 Sonnet**（2025 发布初期）
- 也用 GPT-4o / DeepSeek R1
- **肖弘公开承认"站在巨人肩膀上"**

### Agent 架构
- **Multi-Agent Orchestration**：任务规划 + 执行 + 验证
- **云沙箱环境**：Agent 在隔离虚拟机运行
- **Tool Use**：浏览器、shell、Python、API
- **Memory**：长对话 + 项目状态

### 争议：套壳 or 创新？
- 反方：只是 Claude + orchestration，无自研模型
- 正方：Agent orchestration 是独立价值；产品体验 + 执行力真实

## 五、商业模式

### 订阅
- **Free**：试用
- **Starter**：$39/月
- **Pro**：$199/月
- **Business**：$500+/月

### 2025 ARR 估算
- ~$20-40M（推测，基于用户数和付费率）
- 增长快但基数仍小
- 海外 + 国内混合收入

### 挑战
- **模型 API 成本**：每 Agent 任务 $0.5-5
- **获客成本**：2025-03 爆款之后依赖口碑
- **定价**：国内价格要低（¥50-200），海外$39-200

## 六、竞争与壁垒

### 直接对标
- **Devin**：同为通用 Agent，Devin 更聚焦 Coding
- **ChatGPT Agent / Operator**：同为通用
- **豆包 / Kimi / DeepSeek**：中国大模型 C 端产品
- **Perplexity Deep Research**：同为"研究报告 Agent"

### 壁垒
- **"中国 AI Agent 代表" brand**
- **产品体验** vs 竞品：界面 / 输出结构好
- **全球化能力**：中国公司做海外少见
- **红杉 + 腾讯** 资本支持

### 弱点
- **完全依赖基础模型**（Claude 为主）
- **商业化早期**，ARR 小
- **中美地缘风险**：海外数据合规
- **被 OpenAI Agent 追赶**

## 七、关键风险

- **Anthropic 涨价 / 限流**：套壳模式的结构性风险
- **海外被限制**：数据主权 / TikTok 式被 ban
- **国内合规压力**：Agent 备案 / 监管
- **融资难**：中国公司海外融资难
- **技术跟不上**：自主研发不足

## 八、我的判断

> **我的看法**：
>
> 1. **Manus 是中国 AI 应用最好的"全球化案例"**：ChatGPT / DeepSeek 之外能全球爆火的
> 2. **但"套壳 Claude" 是真实风险**：Claude 4.5 自家 Agent 产品出来后 Manus 差异化受挤压
> 3. **肖弘的产品直觉强** 但**技术深度需要补课**：Agent orchestration 创新有限
> 4. **估值 $500M 合理偏高**：ARR $30M 对应 16x 倍数
> 5. **2026 关键是：能否做出"中国 Agent 能独立" 的技术故事**（比如 fine-tune on DeepSeek / Qwen）
>
> **我可能错在哪里**：
> - Manus 用户规模突破 1000 万 MAU，成为"中国 ChatGPT" 级产品
> - 肖弘团队做出真正 Agent 技术创新（不只是 orchestration）
> - 被腾讯 / 字节以 $3B+ 收购
