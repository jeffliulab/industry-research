# 1X Technologies

> 最后更新：2026-04-24
>
> 1X 是 **挪威 / 美国双总部的人形机器人公司**（原名 Halodi Robotics）——**OpenAI 战略投资**，CEO Bernt Øivind Børnich 技术派。主打产品 **NEO**（2024-10 发布）面向 **消费家庭 / 商用**，2024 融资 $100M+（估值未披露但被认为 $1B+）。

## 一句话定位

1X = **"OpenAI 押注的人形公司"** —— 产品路线独特：**柔性肌腱驱动（tendon-driven）** 而非传统电机 + 齿轮，**安全性 + 亲民** 优先，定位消费家庭。

## 一、公司速览

- **创立**：2014（原 Halodi Robotics，挪威）
- **总部**：挪威 Moss + 美国 Sunnyvale
- **CEO**：Bernt Øivind Børnich
- **员工**：~150
- **估值**：~$1B（2024 推测）
- **主要投资方**：OpenAI（战略）、Tiger Global、EQT Ventures、Samsung、Sandwater

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2014 | 挪威成立 |
| 2022 | 改名 1X |
| 2023-03 | **OpenAI 领投 A2 $23.5M** |
| 2024-01 | B 轮 $100M |
| 2024-10 | **NEO 消费版发布** |
| 2025 | NEO Beta 小范围家庭测试 |

## 三、业务与产品

- **NEO**：消费级家庭人形（折合 $10-20k 预期）
- **EVE**：商用版（仓储 / 零售）
- **目标**：2026-2027 家庭 beta 扩大

## 四、技术路线

- **肌腱驱动**（tendon）：柔性 + 轻量 + 安全
- **端到端 VLA 模型**（与 OpenAI 合作研究）
- **家庭场景优先**：不需要工业级精度

## 五、商业模式

- 未商业化（硬件未大规模出货）
- 定位消费 $10-20k
- 长期类似特斯拉 D2C 路线

## 六、竞争与壁垒

- 对标：Tesla Optimus（家庭方向）、Figure（工业）
- 壁垒：OpenAI 关系 + 肌腱驱动专利
- 弱点：肌腱驱动力量 / 稳定性仍未充分验证

## 七、关键风险

- OpenAI 关系变化（OpenAI 若自建机器人部门 → 1X 被弱化）
- Optimus 家庭版 2026-2027 面市后价格竞争
- 消费安全事故敏感性高

## 八、我的判断

> 1. **1X 是"家庭人形" 最被看好的非 Tesla 玩家**
> 2. **肌腱驱动是有趣的差异化但未被验证**
> 3. **OpenAI 战略投资是护城河也是风险**
> 4. **最可能终局：被 OpenAI 完整收购整合机器人部门**
>
> **我可能错在哪里**：NEO 2026 正式发货后家庭用户喜欢，1X 成为消费人形品牌代表。
