# 小马智行（Pony.ai）

> 最后更新：2026-04-24
>
> 小马智行是 **2016 年彭军 + 楼天城（IOI 金牌）创立的中美双头自驾公司**——中美同步运营（北京 + 广州 + SF + Fremont）。**2024-11 美股 IPO**（$PONY）。**卡车 + Robotaxi 双线**。**Toyota 战略投资**。

## 一句话定位

小马智行 = **"中美同源自驾公司"** —— 彭军 / 楼天城 **Google / Baidu 硅谷技术血统** + 广州 / 北京商业化基地。**2024 IPO 后面临股价压力**，但**Toyota 合作 + 卡车线路**提供支撑。

## 一、公司速览

- **创立**：2016
- **总部**：北京 + Fremont CA
- **创始人**：彭军（CEO）+ 楼天城（CTO）
- **员工**：~1,000
- **上市**：**$PONY**（2024-11 纳斯达克 IPO）
- **市值**：$5-7B（2025 波动）

## 二、历史

| 时间 | 里程碑 |
|---|---|
| 2016 | 成立 |
| 2020 | Toyota 领投 $462M |
| 2022 | 北京全区 Robotaxi |
| 2024 | 丰田合作生产 Robotaxi |
| **2024-11** | **纳斯达克 IPO** |

## 三、业务

- **Robotaxi**：北京 / 广州 / 上海 / SF 测试
- **Robotruck**：货运 / 高速
- **Virtual Driver**：技术平台

## 四、技术路线

- 模块化 + 端到端融合
- LiDAR + 摄像头
- Toyota / Aurora 合作车型

## 五、商业模式

- Robotaxi per-ride
- 卡车 per-mile
- 2024 营收 ~$90M

## 六、竞争与壁垒

- 对标：Waymo、萝卜、文远
- 壁垒：**中美双基地 + Toyota 合作**
- 弱点：**规模 < Waymo / 萝卜**、**中美地缘**

## 七、我的判断

> 1. **小马是中美双线博弈的代表**
> 2. **IPO 后股价压力大** 但**Toyota 撑场**
> 3. **卡车业务稳定现金流**
> 4. **大概率与文远 / 萝卜分三足格局**
>
> **我可能错在哪里**：中美地缘紧张迫使拆分两地业务。
