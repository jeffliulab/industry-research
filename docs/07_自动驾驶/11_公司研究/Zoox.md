# Zoox

> 最后更新：2026-04-24
>
> Zoox 是 **2014 年 Tim Kentley-Klay + Jesse Levinson 创立的 Robotaxi 公司**——2020 年 **被 Amazon 以 $1.2B 收购**。**自主设计 L4 Robotaxi 车型**（无方向盘 / 对坐设计）。2024-2025 **Las Vegas + SF 测试扩展**。

## 一句话定位

Zoox = **"Amazon 的 Robotaxi 押注"** —— 唯一一家 **从 0 自主设计 L4 车型** 的公司（非改装）。**规模 + 商业化 落后 Waymo / 萝卜**，但车型设计独特。

## 一、公司速览

- **创立**：2014
- **总部**：Foster City, CA
- **所有者**：**Amazon**（2020 $1.2B 收购）
- **CEO**：Aicha Evans
- **员工**：~2,000

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2014 | 成立 |
| 2018 | 路测 |
| **2020-06** | **Amazon $1.2B 收购** |
| 2022-02 | 自主设计车型公开 |
| 2024 | Vegas + SF 测试 |
| 2025 | 商业试点计划 |

## 三、产品

- **Zoox Robotaxi**：4 座对坐 + 无方向盘
- **双向行驶**：无需调头

## 四、技术路线

- 混合 LiDAR + 摄像头
- 自研全栈（车 + 软 + 硬）

## 五、商业模式

- 规划 Robotaxi 付费运营
- Amazon 可能整合送货 / 乘车

## 六、我的判断

> 1. **Zoox 是 Waymo 之外最特色 Robotaxi**
> 2. **Amazon 整合 可能颠覆** 但节奏慢
> 3. **2026 实际商用** 是关键验证
>
> **我可能错在哪里**：自主车型成本太高，Amazon 放弃。
