# 百度萝卜快跑

> 最后更新：2026-04-24
>
> 百度萝卜快跑是 **中国 Robotaxi 规模化领先产品**——**2022 武汉起步**，2024 **武汉 L4 规模化运营**（500+ 辆车，**每次出行 ¥4-18** 比打车便宜 80%）。百度 Apollo 长期投入下的商业化落地。**2024-2025 最大争议**：武汉出租车司机罢工抗议。

## 一句话定位

萝卜快跑 = **"中国 Robotaxi 商业化第一"** —— 政府主导 + 百度技术 + 武汉配合。**每公里成本 / 收入**已接近中国出租车水平，**2026-2027 目标盈亏平衡**。**监管友好 + 补贴** 是结构优势。

## 一、业务速览

- **隶属**：百度 Apollo
- **总部**：北京 + 武汉
- **负责人**：王云鹏（Apollo + 智能驾驶事业群总裁）
- **运营车辆**：~1,000（武汉 + 北京 + 广州）
- **累计订单**：1000 万 + （2025-Q1）

## 二、历史

| 时间 | 里程碑 |
|---|---|
| 2017 | Apollo 计划启动 |
| 2022-08 | 武汉商用起步 |
| 2024-04 | **武汉 500+ 辆规模** |
| 2024-07 | **司机罢工抗议** 事件 |
| 2024 | 北京 / 广州 / 上海扩展 |
| 2025 | 目标 10 城市 2000+ 车 |

## 三、产品

- **RT6**（百度自研 Robotaxi 车型）：$30k 目标成本
- **小程序 + App** 打车
- **多个品牌合作** 车型

## 四、经济学

- **每次出行 ¥4-18**（武汉）
- **每公里成本 ¥3-5**
- **利用率 40%**
- **2026-2027 目标盈亏平衡**

## 五、监管 / 政策

- **武汉政府全力支持**（东风汽车合作）
- **北京 / 深圳 / 广州** 跟进
- **2024-07 武汉司机罢工**：社会影响

## 六、竞争与壁垒

- 对标：小马智行、文远知行（中国）
- 壁垒：**百度 Apollo 技术 + 政府关系 + 规模**
- 弱点：**百度集团 AI 投入 波动**

## 七、我的判断

> 1. **萝卜快跑是中国 Robotaxi 无可争议第一**
> 2. **2026-2027 盈亏平衡可能早于 Waymo**（成本结构 + 补贴）
> 3. **社会争议**（出租车司机）会持续
> 4. **终局：IPO 或作为百度旗舰业务**
>
> **我可能错在哪里**：社会阻力让政府收紧政策，萝卜停滞。
