# Motional

> 最后更新：2026-04-24
>
> Motional 是 **现代汽车 + Aptiv 50/50 合资的自驾公司**——2020 年由 **Aptiv Autonomous（原 nuTonomy）+ 现代 Autonomous** 合并。**拉斯维加斯 / 波士顿** L4 测试。**与 Uber / Lyft 合作 Robotaxi 试点**。2024 遇资金压力，**现代 2024-05 追加 $1B 投资**。

## 一句话定位

Motional = **"现代汽车的自驾押注"** —— 与 Waymo / Cruise / Zoox 同层级玩家，但规模和声量小。**拉斯维加斯 Robotaxi 已运营**，**Boston 研发总部**。**2024 融资压力 + 裁员 300+** 是 2025 挑战。

## 一、公司速览

- **创立**：2020（Aptiv + Hyundai 合资）
- **总部**：Boston
- **CEO**：Karl Iagnemma
- **员工**：~1,000（裁员后）
- **估值**：~$4B
