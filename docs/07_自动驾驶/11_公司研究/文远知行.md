# 文远知行（WeRide）

> 最后更新：2026-04-24
>
> 文远知行是 **2017 年广州起家的中国 L4 自驾公司**——**韩旭（前百度 L4 总监）+ 李岩** 创立。**2024-10 美股 IPO**（$WRD）。**产品覆盖最广**：Robotaxi、Robobus（巴士）、Robovan（小巴）、Robostreet Cleaner（扫地）。**迪拜 + 沙特扩展积极**。

## 一句话定位

文远 = **"产品最多元的中国 L4 公司"** —— 不只 Robotaxi，还做**自动清洁车 / 巴士 / 货运**。**海外扩展最积极**（迪拜、新加坡、沙特、阿布扎比）。**2024 IPO 后估值 $5B 左右**。

## 一、公司速览

- **创立**：2017
- **总部**：广州 + SF
- **创始人 / CEO**：韩旭
- **员工**：~1,000+
- **上市**：**$WRD**（2024-10）
- **市值**：~$5B

## 二、历史

| 时间 | 里程碑 |
|---|---|
| 2017 | 成立 |
| 2018 | 广州路测 |
| 2020 | Robotaxi 商用 |
| 2023-07 | 阿布扎比合作 |
| **2024-10** | **美股 IPO** |

## 三、产品

- **Robotaxi**：广州 + 多城
- **Robobus**（小巴）：机场 + 园区
- **Robovan**（货运 + 小巴）
- **Robostreet Cleaner**（清洁车）：已量产 + 2000+ 辆售出

## 四、技术路线

- 模块化 + 端到端
- LiDAR + 摄像头

## 五、商业模式

- 硬件销售（清洁车）+ Robotaxi per-ride
- 2024 营收 $50M+（清洁车贡献大）

## 六、竞争与壁垒

- 对标：小马、萝卜、Waymo
- 壁垒：**多产品线 + 海外（中东）+ 清洁车商用**
- 弱点：**规模小 + 中美地缘**

## 七、我的判断

> 1. **文远是"产品多元"聪明策略**——不只 Robotaxi
> 2. **清洁车现金流** 支撑自驾研发
> 3. **中东市场** 是独特优势
> 4. **长期稳定第二梯队**
>
> **我可能错在哪里**：清洁车业务被快速商品化侵蚀。
