# Aurora Innovation

> 最后更新：2026-04-24
>
> Aurora 是 **2017 年 Chris Urmson（Google SDC 前技术负责人）+ Sterling Anderson（Tesla 前 AP 负责人）+ Drew Bagnell（Uber ATG）**创立——**卡车自驾专注**。**2021 SPAC 上市**（$AUR）。**2024-10 Fort Worth → El Paso 商业运营**是**全球首个 L4 卡车商用**。

## 一句话定位

Aurora = **"卡车自驾事实领导者"** —— 三巨头创始人 + 务实卡车路线 + 2024 首个商用。**比乘用车 Robotaxi 技术难度低**（高速单一场景）但**经济 ROI 更清晰**。

## 一、公司速览

- **创立**：2017
- **总部**：Pittsburgh + SF
- **创始人**：Chris Urmson、Sterling Anderson、Drew Bagnell
- **员工**：~1,800
- **上市**：**$AUR**（2021 SPAC）
- **市值**：$10-15B 波动（2025）

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2017 | 成立 |
| 2020 | 收购 Uber ATG |
| 2021 | **SPAC IPO $13B** |
| 2023 | 扩展卡车试点 |
| **2024-10** | **Fort Worth → El Paso 商业运营**（首个 L4 卡车）|
| 2025 | 扩展线路 |

## 三、业务与产品

- **Aurora Driver**：自驾 L4 系统
- **合作伙伴**：Volvo Trucks、PACCAR、Uber Freight、FedEx
- **乘用车 Robotaxi**（Toyota 合作）2024 年暂缓

## 四、技术路线

- **LiDAR + 摄像头 + 雷达**
- 自研 FirstLight LiDAR
- 高速公路场景优先

## 五、商业模式

- **Per-mile 付费** （按里程收费）
- **2024-2025 初始收入 ~$100M**
- 未盈利但路径清晰

## 六、竞争与壁垒

- 对标：Kodiak、Plus（原 Plus.ai）、TuSimple（破产）
- 壁垒：**三创始人 brand + 上市融资 + PACCAR / Volvo 合作**
- 弱点：**股价压力 + 竞争激烈**

## 七、我的判断

> 1. **Aurora 是卡车自驾商业化领导者**
> 2. **2026-2028 盈亏平衡** 期望
> 3. **股价长期看技术 + 规模**
> 4. **终局：作为 Volvo / PACCAR 合作核心** 稳定
>
> **我可能错在哪里**：公路事故 / 监管变化导致 Aurora 重挫。
