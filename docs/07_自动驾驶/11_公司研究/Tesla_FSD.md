# Tesla FSD

> 最后更新：2026-04-24
>
> Tesla FSD（Full Self-Driving）是 **Tesla 2016 起的自驾项目**——**规模最大 L2+ 辅助驾驶** + **2024-2025 转向 Robotaxi**（Cybercab 2024-10 发布）。**FSD v12 2024-03 转端到端** 是里程碑。用户 **500 万 +**，**订阅制 $99/月 或一次买断 $8k**。**Elon Musk 反复承诺 L4 但迟迟未达**。

## 一句话定位

Tesla FSD = **"L2+ 规模最大 + Robotaxi 野心最大"** —— 卖车 + 软件 + Robotaxi 三位一体。**端到端 FSD v12 + Cybercab + Optimus** 是 Elon 的 "**物理 AI** "大布局。

## 一、公司速览

- **隶属**：Tesla Inc
- **CEO**：Elon Musk
- **FSD 团队规模**：~300（Dojo + 算法 + Data）
- **Tesla 市值**：~$1.3T（2025 波动）

## 二、FSD 版本演化

| 版本 | 时间 | 特点 |
|---|---|---|
| AP 1.0 | 2014 | Mobileye |
| AP 2.0 | 2016 | 自研开始 |
| AP 3.0 / HW3 | 2019 | 自研 SoC |
| FSD Beta v10 | 2022 | 有限用户 |
| FSD Beta v11 | 2023 | 公开扩大 |
| **FSD v12** | 2024-03 | **端到端** |
| FSD v13 | 2024-10 | 改进 |
| HW4 / AI4 | 2023 → 部署 |
| **HW5 / AI5** | 2024-10 Cybercab 发布 |

## 三、产品

- **FSD Beta 订阅** $99/月 或 $8,000 一次买断
- **Cybercab**（2024-10 发布）：$30k 目标 + Robotaxi
- **Optimus**：人形（共享 FSD 大脑）

## 四、技术路线

- **纯视觉**（无 LiDAR，激进）
- **端到端神经网络**（v12 起）
- **Dojo 超算** + HW5 车载芯片
- **数据 车队**：数十亿 km

## 五、商业模式

- **FSD 订阅 / 买断**：估算 **$3B+ 2024 收入**
- **Cybercab Robotaxi 未来**：2026 起运营
- **Robotaxi 平台 70/30 分成**（车主 + Tesla）

## 六、用户规模

- **500 万 +** Tesla 车主可用 FSD
- **订阅转化率** ~20%
- **累计 FSD 英里**：数十亿

## 七、监管挑战

- NHTSA 多次调查 Autopilot
- **2023-12 大规模召回**（软件更新）
- 加州起诉 Tesla "FSD 误导" 广告
- **2024-2025 持续监管压力**

## 八、我的判断

> 1. **Tesla FSD 仍是 L2+ 最大规模**
> 2. **Elon 承诺 L4** 一再推迟——但**技术是真的进步**
> 3. **Cybercab 2026 能否出货** 是关键
> 4. **纯视觉路线** 仍是最有争议决策
>
> **我可能错在哪里**：Cybercab 不能量产；纯视觉方案碰安全天花板。
