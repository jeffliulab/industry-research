# 百度 Apollo

> 最后更新：2026-04-24
>
> 百度 Apollo 是 **百度 2017 年启动的自驾开放平台**——**中国自驾行业的 Linux**。2017-2020 **开源阶段**吸引几百家车厂 / 零部件 + 高校 + 研究。**2021 起转向商业化**：**Apollo Go（萝卜快跑）+ 与吉利 / 广汽 / 岚图合作**。

## 一、产品定位

百度 Apollo = **"中国自驾开源 / 商业双轨"** —— **Apollo 开源平台** 仍在维护（版本 8.0 / 9.0）；**Apollo 商用** 以 **萝卜快跑**为旗舰。**王云鹏负责**（百度自驾总裁）。

## 二、核心组件

### Apollo 开源平台
- **GitHub star 20,000+**
- **模块化全栈**：感知 / 规划 / 控制 / 地图 / HMI
- **CyberRT**：机器人操作系统（替代 ROS）
- **9.0 版本（2024）**：改进 端到端 + 仿真

### Apollo 商用
- **Apollo RT 系列** Robotaxi 车型
- **萝卜快跑 App** 
- **Apollo D** L2+ 方案（给 OEM）

## 三、合作伙伴

### 车厂
- **吉利 / 极越**
- **广汽埃安**
- **东风**（武汉合作）
- **岚图**

### 供应链
- 地平线 + 英伟达 芯片
- 禾赛 / 速腾 LiDAR
