# Robotaxi 经济学：Waymo · 萝卜快跑 · Zoox

> 最后更新：2026-04-24
>
> Robotaxi 2024-2026 进入 **规模化关键窗口**——**Waymo 每周 15 万次付费乘车**（2025-Q1，旧金山 + Phoenix + LA）、**百度萝卜快跑 2024 武汉规模化**、**Cruise 2023 事故后停运 → 2025 重启**。**单位经济学**：每公里成本 vs 收入仍然亏损但收敛中。

## 一句话结论

Robotaxi 2026 **"技术可行 + 经济仍未盈利"**—— **Waymo 每公里成本 $5-8 vs 收入 $3-5**，仍亏损。**规模 + 硬件成本下降 + 运营效率提升** 是唯一路径。**Tesla FSD 2025 Robotaxi 发布**（Cybercab）是最大变量。

## 三条关键要点

1. **Waymo 是全球 Robotaxi 领跑者**：周 15 万次付费 + 3 城扩展 + 每年成本下降
2. **萝卜快跑武汉案例**：**每次出行 ¥4-18** 比出租车低 80%（但亏损）
3. **Tesla Cybercab**（2024-10 发布）：$30k 目标 + FSD 自主，2026 起量产

## 单位经济学（粗估）

### Waymo
- **每公里成本**：~$5-8
- **每公里收入**：~$3-5
- **利用率**：30-40%
- **目标：2027-2028 盈亏平衡**

### 萝卜快跑（武汉）
- **每公里成本**：~¥3-5
- **每公里收入**：~¥1-3
- 政府补贴 + 车队规模支撑
- **盈亏平衡 2026-2027**

## 成本结构

```
硬件（车辆 + 传感器）：30-40%
运营（充电 / 维护 / 清洁）：20-30%
软件（AI 模型 + 云）：10-15%
保险：15-20%
监控 / 远程接管：10-15%
```

## 关键变量降本

- **硬件**：LiDAR 从 $10k → $1k（5 年）
- **车辆**：Cybercab $30k vs 目前 Robotaxi $200k+
- **保险**：规模 + 安全证明 → 降费
